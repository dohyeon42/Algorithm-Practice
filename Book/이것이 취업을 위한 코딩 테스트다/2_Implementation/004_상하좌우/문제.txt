여행가 A는 N * N크기의 정사각형 공간에 있다. 이 공간은 1 * 1크기의 정사각형으로 나누어져 있다.
가장 왼족 위 좌표는 (1,1)이며, 가장 오른쫀 아래 좌표는 (N, N)이다.
여행가 A는 상, 하, 좌, 우로 이동할 수 있고, 시작은 항상 (1,1)이다.

계획서는 하나의 줄에 공백 기준으로 L, R, U, D중 하나의 문자가 반복적으로 적혔다.
의미는 아래와 같다.
    L : 왼쪽으로 한 칸 이동
    R : 오른쪽으로 한 칸 이동
    U : 위쪽으로 한 칸 이동
    D : 아래쪽으로 한 칸 이동
이때 여행가 A가 N*N의 정사각형을 벗어나는 움직임은 무시한다.

입력 조건:
    첫째 줄에 공백구분으로 N이 입력된다.
    (
        1 <= N <= 100 (N은 자연수)
    )
    둘째 줄에 여행가 A가 이동할 계획서의 내용이 주어진다.
    (
        1 <= 이동횟수 <= 100
    )

출력 조건:
    첫째 줄에 여행가 A의 최종 도착 좌표 (X,Y)를 공백으로 구분하여 출력하시오.

입력 예시:
    5
    R R R U D D
출력 예시:
    3 4
