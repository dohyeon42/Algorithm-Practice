숫자 카드 게임은 여러 개의 숫자 카드 중에서 가장 높은 숫자가 쓰인 가드 한 장을 뽑는 게임.
룰:
    1. 숫자가 쓰인 카드가 N * M로 놓인다.(N은 행 개수, M은 행 개수)
    2. 먼저 뽑을 카드가 있는 행을 선택
    3. 선택한 행에 카드중 가장 숫자가 낮은 카드를 뽑는다.
    4. 처음에 카드를 고를 행을 고를때 이후에 해당 행에서 가장 수가 낮은 카드를 뽑을 것을 고려하자

카드들이 N * M으로 놓일때 게임의 룰에 맞게 카드를 뽑는 프로그램을 만드시오.

입력 조건:
    첫째 줄에 숫자 카드들이 놓인 행의 개수 N과 열의 개수 M이 공백 구분 입력된다(1 <= N,M <= 100)
    둘째 줄부터 N개의 줄에 걸쳐 각 카드에 적힌 숫자가 주어진다. 숫자는 1 이상, 10000이하의 자연수이다.

출력 조건:
    첫째 줄에 룰에 맞게 선택된 카드에 적힌 숫자를 출력한다.

입력 예시1:
    3 3
    3 1 2
    4 1 4
    2 2 2

출력 예시2:
    2

입력 예시2:
    2 4
    7 3 1 8
    3 3 3 4

출력 예시2:
    3
